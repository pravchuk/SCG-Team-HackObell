Genie
	- By team HackoBall
This is an application for the users of SG Bank, which allows users to enter their respective wishlists.
Depending on the trends in the wishlist, the bank will create special loans and emis, to increase the customer base.
Providing better deals to customers on important events like birthdays rather than the conventional birthday message, will increase loyality. 
This will also give banks a clear idea on which company's products is in demand. Hence they can invest in that company and make better profits. 

Contains source code.
To deploy, download and install cordova, node.js and ionicFramework. 
Then run the code via cordova and deploy on phone of your choice.
